설치 및 실행방법

1. < > Code 를 클릭해서 Download ZIP을 눌러서, ZIP 파일로 다운 받는다.

2. 다운 받은 파일의 압축을 풀고, Bansawee-main 폴더 안의 main.py를 실행한다.

3. VS code를 이용해서 실행할 경우, main.py를 클릭하면 바로 VS code가 열리고, 실행하면 된다.

3-1. VS code에서 오른쪽 위 삼각형 실행 버튼을 누르면 실행된다. 버튼이 없다면, 왼쪽 세로바에 Run&Debug를 누르면 오른쪽 위에 버튼이 생긴다.

4. 터미널에서 실행할 경우, python을 미리 설치해야한다. 

4-1. 설치한 파일의 경로를 복사하여 터미널에서 "cd 파일경로" 를 입력한다. 이때, main.py 파일이 보이는 화면 상태의 파일 경로를 입력해야한다.

4-2. 터미널에서 "python main.py"를 입력한다.



게임 방법

1. 주사위를 던져 뱀형 보드판을 이동합니다.

2. '전투 칸'에서는 주사위 기반 전투가 발생합니다.

2-1. 플레이어의 기본 능력치는
체력 - 100
공격력 - 10
라이프 - 3 
입니다.

2-2. 주사위를 굴려 나온 숫자에 따라 능력치가 증가할 수도 있고, 감소할 수도 있습니다.

2-3. 승리 시, 라이프가 감소하지 않습니다. 패배 시, 라이프가 1 감소합니다.

3. '아이템 칸'에서는 전투에서 쓰는 주사위 관련 아이템을 획득할 수 있습니다.

4. "미니게임 칸"에서는 라이프 회복 또는 랜덤 아이템을 획득할 수 있습니다.

5. 라이프가 0이 되면 게임이 종료됩니다.

6. 보드판의 마지막 칸에 도착하여 보스를 물리치면 승리합니다.
